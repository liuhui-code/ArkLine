# ArkLine Third-Party Notices

## Planned Runtime Dependencies

- Tauri v2
- Rust standard ecosystem crates
- React
- Vite
- CodeMirror 6 core packages
- Vitest

## Planned Optional Tooling

- `git.exe`
- `rg.exe`
- `@arkts/language-server`
- user-configured lint and format executables

## Distribution Constraint

Do not bundle HarmonyOS, HMS, or OpenHarmony SDK assets into ArkLine. Detect or
configure user-installed toolchains instead.
